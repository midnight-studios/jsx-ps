To run the script in Illustrator and Photoshop.


1. Put file 'SUIB.jsxbin' in the directory '/[program folder]/Presets/Scripts/'. 

If this folder does not exist - create it.


2. Put file 'ScriptUI-builder_v2.3 Loader.jsx':

- for Photoshop CS4/CS5: in the language-localized directory '/[program folder]/Presets/Scripts/'.


- for Illustrator CS4/CS5: in the language-localized directory '/[program folder]/Presets/[locale]/Scripts/'.


3. Restart Illustrator/Photoshop and run the script from menu 'File/Scripts'.
